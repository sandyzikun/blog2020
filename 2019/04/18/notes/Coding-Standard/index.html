O. Code Style的三大基本问题

  1. 前后大括号是否换行
    [强制] 代码块的前大括号不换行, 后大括号需要换行。

  2. 缩进使用Space还是Tab
    [强制] 所有缩进一律使用Space*4, 禁止Space*2或长度不定的Tab。

  3. 单句流程控制语句是否使用大括号
    [强制] 单句流程控制语句使用大括号。
